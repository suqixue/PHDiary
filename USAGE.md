## 快速开始

### 作为普通用户

#### 第一步：打开应用
- 在浏览器中打开 `研究生自我管理系统.html`
- 第一次打开会自动进入 **演示版本**（1天试用期）

#### 第二步：设置个人信息
1. 点击左侧的**头像**区域
2. 修改以下信息：
   - 姓名
   - 学号/工号
   - 头像（点击上传）
   - 背景主题（自定义或预设）

#### 第三步：开始使用
开始记录你的日常数据：
- 💼 **工作打卡**：点击"开始工作"
- 😊 **心情记录**：点击心情表情
- 😤 **压力管理**：调整压力滑块
- ✅ **任务管理**：添加待办事项

#### 第四步：升级到专业版
如需使用所有功能（主题自定义、数据导出等）：
1. 点击 **"⭐ 升级"** 按钮或 **"✓ 已激活"** 按钮（如果已激活）
2. 输入 **用户ID** 和 **许可证密钥**
3. 点击 **"激活"**
4. 激活成功！所有功能解锁

## 指标计算说明

本系统采用科学、专业的指标计算方法，各项指标均有明确的计算逻辑和学术支撑。

### 工作效率指数 (0-100)

#### 计算公式
```
工作效率 = (日工作时长 / 基准时长) × 50 + (工作连续性评分) × 50
```

#### 详细说明
- **日工作时长部分** (0-50分)
  - 基准时长：8小时
  - 工作时长 ÷ 8小时 × 50分
  - 超过12小时按12小时计算（避免过度工作奖励）

- **工作连续性部分** (0-50分)
  - 统计当天最长连续工作时长
  - 未中断工作时间越长，得分越高
  - 参考番茄工作法（25分钟为基础单位）
  - 连续时间满4小时为满分

#### 出处与参考
- 基于《时间管理的艺术》(The Art of Time Management) 理论
- 参考高生产力研究：Alan Lakein 时间管理象限法
- 连续性参考：深度工作理论 (Deep Work) - Cal Newport

---

### 专注力指数 (0-100)

#### 计算公式
```
专注力指数 = (最长连续工作时长 / 目标时长) × 100
```

#### 详细说明
- **最长连续工作时长**：当天未中断的工作时长
- **目标时长**：4小时（基于研究推荐）
- 达到4小时为100分满分
- 未达到按比例计算

#### 参考理论
- **番茄工作法**：建议25分钟工作 + 5分钟休息
  - 连续工作4小时 = 8个番茄周期
  - 符合人体专注力规律

- **时间生物学研究** (Chronobiology)
  - 人类最佳专注时间：90-120分钟
  - 推荐工作周期：4小时（含2-3次短休息）
  - 超出4小时效率下降明显

---

### 身心指数 (0-100)

#### 计算公式
```
身心指数 = 工作效率 × 0.3 + 健康状态 × 0.4 + 心理状态 × 0.3
```

#### 详细说明

**工作效率部分** (30%)
- 使用上述"工作效率指数"
- 反映学术生产力

**健康状态部分** (40%)
- 运动评分：每周运动 ≥3次得30分，否则递减
- 饮水评分：每日2000ml得20分
- 咖啡因评分：每日 ≤400mg得20分（>400mg开始扣分）
- 睡眠评分：根据熬夜情况判断（10最高）

**心理状态部分** (30%)
- 心情评分：5档心情中性比例
  - 😄😊（积极）：各30分
  - 😐（中立）：20分
  - 😔😫（消极）：各10分
- 压力评分：0-100实时滑块
  - 0-20：20分
  - 20-60：15分
  - 60-80：10分
  - 80-100：5分

#### 参考来源
- **世界卫生组织 WHO 健康定义**：身体、心理、社会适应完好
- **学生健康与学术成就研究**：运动与学习成绩正相关
- **心理健康指数模型** (Mental Health Index)
- **生活质量测量表** (SF-36)

### 心情评分

#### 记录标准
```
😄 非常开心    - 感到快乐、满足、成就感
😊 开心        - 感到不错、顺利、积极
😐 还行        - 平常、中立、无特殊情绪
😔 难受        - 感到沮丧、疲惫、失望
😫 非常烦      - 感到极度困扰、焦虑、压力大
```

#### 用途
- 追踪长期心理趋势
- 分析心情与工作/压力/健康的关联
- 识别情绪规律和诱发因素

### 压力指数 (0-100)

#### 等级划分
```
🟢 绿色 (0-20)   - 轻松、放松状态
🟡 黄色 (20-60)  - 正常、可控压力（有挑战）
🟠 橙色 (60-80)  - 高压力、需要调整
🔴 红色 (80-100) - 极高压力、需要紧急干预
```

#### 参考指标
基于 **Stress Management Society** (SMS) 压力等级标准：
- 低压力：利于学习和工作
- 中等压力：最优表现区间（Yerkes-Dodson规律）
- 高压力：影响睡眠和健康
- 极高压力：需要心理健康干预

### 咖啡因摄入安全标准

#### 标准值
- **安全上限**：每日400mg（FDA推荐）
- **敏感人群**：每日≤200mg
- **儿童/青少年**：不建议摄入

#### 常见饮品咖啡因含量
```
浓缩咖啡 (1杯/30ml)      - 63-72mg
美式咖啡 (1杯/240ml)    - 95-200mg
拿铁咖啡 (1杯/240ml)    - 75-150mg
卡布奇诺 (1杯/240ml)    - 75-150mg
黑茶 (1杯/240ml)        - 40-70mg
绿茶 (1杯/240ml)        - 25-50mg
红茶 (1杯/240ml)        - 25-50mg
可乐 (1罐/355ml)        - 34-46mg
```

#### 摄入过量警告 (>400mg)
- 可能导致：失眠、焦虑、心悸
- 系统提醒用户控制摄入
- 建议下午3点后避免咖啡因

#### 参考
- **美国食品药品管理局 (FDA)** 咖啡因安全指南
- **欧洲食品安全局 (EFSA)** 咖啡因风险评估报告

### 运动卡路里消耗估算

#### 计算基础
```
消耗卡路里 = MET值 × 体重(kg) × 运动时间(小时)
```

#### 常见运动MET值
```
静坐          - 1.0 MET
散步 (3km/h)  - 2.8 MET
慢走 (5km/h)  - 3.5 MET
快走 (6km/h)  - 4.3 MET
慢跑 (8km/h)  - 8.0 MET
篮球          - 7.0 MET
足球          - 10.0 MET
羽毛球        - 8.0 MET
网球          - 8.0 MET
乒乓球        - 4.0 MET
游泳          - 8.0 MET
骑自行车      - 7.5 MET
```

#### 计算示例
```
75kg体重的人慢跑1小时
= 8.0 MET × 75kg × 1h = 600卡路里
```

#### 参考
- **美国体育医学学会 (ACSM)** MET值表
- **运动生理学研究** (Exercise Physiology)
- **健身指导标准** (Fitness Standards)

### 成就徽章系统

#### 徽章类型与条件

**工作相关** 
- 7天连续打卡：连续7天每天工作≥1小时
- 月均8小时：月累计≥240小时（平均8小时/日）
- 连续专注：单次工作≥4小时不中断
- 工作狂人：单日工作≥12小时

**健康相关**
- 运动达人：周运动≥3次
- 饮水大师：周均每日≥2000ml
- 咖啡因控制：周均每日≤400mg
- 早起鸟：连续5天6:00前开始工作
- 夜猫子：连续5天23:30后还在工作

**科研相关**
- 论文马拉松：论文类科研≥50小时/月
- 实验狂人：实验类科研≥50小时/月
- 文献收集家：文献阅读≥100条记录

**生活相关**
- 打卡达人：连续30天使用系统
- 自律大师：所有指标优秀≥20天
- 健康卫士：身心指数≥80分≥15天

#### 参考
基于行为心理学中的：
- **习惯形成理论**：21天形成习惯，66天成为自动行为
- **强化学习**：即时反馈提升动力
- **目标设定理论** (Goal Setting Theory) - Locke & Latham

### 本地文件 LocalStorage 的限制

由于**浏览器安全策略限制**，通过 `file://` 协议打开本地HTML文件时，LocalStorage的行为存在不确定性。根据MDN官方文档：

> "For documents loaded from `file:` URLs, the requirements for `localStorage` behavior are undefined and may vary among different browsers...So it's possible that browsers may change their `file:` URL handling for `localStorage` at any time."

### 可能的风险

| 风险 | 说明 |
|-----|------|
| 📦 浏览器更新 | 浏览器更新后可能改变file:// URL的LocalStorage处理方式 |
| 🔒 隐私设置 | 某些浏览器配置或隐私模式可能禁用LocalStorage |
| 🔄 跨浏览器不一致 | 不同浏览器对本地文件的LocalStorage支持差异 |
| 🧹 自动清理 | 浏览器清理缓存时可能删除LocalStorage数据 |
| 🌐 系统更新 | 操作系统更新或清理工具可能清除浏览器文件 |

### 🔥 强烈建议：定期备份数据

#### 每周备份一次
```
左侧菜单 → "💾 保存位置" → "📥 导出数据"
→ 自动下载 PHDiary_备份_日期.json
→ 保存到云盘、外部驱动器或U盘
```

#### 重要时刻立即备份
- ✅ 重要数据变化后
- ✅ 周报生成后
- ✅ 新增成就解锁后
- ✅ 长期数据分析完成后

#### 快速恢复
```
左侧菜单 → "💾 保存位置" → "📤 导入数据"
→ 选择备份文件 PHDiary_备份_XXX.json
→ 点击导入，所有数据立刻恢复
```

### Q: 为什么需要许可证？我以为是免费的？

- ✅ 所有功能完全免费，无广告、无付费购买
- ✅ 首次使用享受 **1天免费试用期**
- ⏱️ 1天后需要许可证激活才能继续使用
- 📧 许可证免费申请，邮箱：suqi.xue@hdu.edu.cn

### Q: 许可证怎么获取？
**A:** 很简单，只需三步：
1. **发送邮件** 到 suqi.xue@hdu.edu.cn
   - 邮件内容：说明你的用途（学习/研究/个人使用等）
   - 可附带你想要的用户ID（邮箱、学号等）

2. **接收回复** - 管理员会在邮件中发送：
   - 你的用户ID
   - 许可证密钥（24个字符）

3. **在应用中激活**：
   - 点击"⭐ 升级"输入用户ID和密钥
   - 激活成功，继续使用！

---

### Q: 1天试用期过后会发生什么？
**A:** 1天试用期自动到期，系统会：
- ❌ 自动禁用所有数据访问
- ⚠️ 显示"许可证已过期"警告
- 🔒 无法查看和编辑任何数据
- 邮箱获取许可证后立即恢复可用

**数据安全：** 你的数据仍完好存储，激活许可证后可立即恢复使用。

### Q: 许可证激活失败，提示"用户ID不匹配"？
**A:** 检查以下几点：
1. **用户ID是否完全相同**
   - 邮件中的ID
   - 应用中输入的ID
   - 必须完全一致（大小写敏感）

2. **许可证密钥是否正确**
   - 确保复制完整（24个字符）
   - 格式：XXXX-XXXX-XXXX-XXXX-XXXX

3. **许可证是否过期**
   - 检查邮件中的有效期
   - 重新申请新的许可证

如仍有问题，请回复邮件说明情况，管理员会帮助处理。

### Q: 可以在多个设备上使用同一个许可证吗？
**A:** 可以！许可证支持多设备激活：
1. **首次激活**：在设备A输入许可证
   - 绑定设备A的指纹
   
2. **使用另一个设备**：去设备B激活相同许可证
   - 可能提示"此许可证已在其他设备激活"
   - 点击"在此设备激活"确认
   - 将指纹切换到设备B

3. **同时使用多设备**：
   - 设备A和B都可以使用
   - 天数共享（一个许可证）
   - 推荐使用设备确认按钮避免冲突
