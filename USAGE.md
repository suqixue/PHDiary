## 快速开始

### 作为普通用户

#### 第一步：打开应用
- 在浏览器中打开 `研究生自我管理系统.html`
- 第一次打开会自动进入 **演示版本**（1天试用期）

#### 第二步：设置个人信息
1. 点击左侧的**头像**区域
2. 修改以下信息：
   - 姓名
   - 学号/工号
   - 头像（点击上传）
   - 背景主题（自定义或预设）

#### 第三步：开始使用
开始记录你的日常数据：
- 💼 **工作打卡**：点击"开始工作"
- 😊 **心情记录**：点击心情表情
- 😤 **压力管理**：调整压力滑块
- ✅ **任务管理**：添加待办事项

#### 第四步：升级到专业版
如需使用所有功能（主题自定义、数据导出等）：
1. 点击 **"⭐ 升级"** 按钮或 **"✓ 已激活"** 按钮（如果已激活）
2. 输入 **用户ID** 和 **许可证密钥**
3. 点击 **"激活"**
4. 激活成功！所有功能解锁

## 用户使用指南

### 工作打卡与时间管理

#### 开始/结束工作
```
点击 "▶️ 开始工作" → 开始计时
工作中时显示 "⏸️ 暂停工作" / "⏹️ 结束工作"
点击 "⏹️ 结束工作" → 保存工作时段
```

#### 查看工作历史
- 左侧导航栏 → "📊 数据统计" 
- 选择日期范围查看工作时段详情
- 可编辑或删除错误的时段记录

#### 工作统计
- 每日工作总时长自动计算
- 工作效率指数基于工作时长和连续性
- 可生成周报查看工作趋势

### 心情与压力管理

#### 记录心情
```
5档心情：
😄 非常开心  →  点击直接记录
😊 开心      →  自动标记时间
😐 还行      →  存储到历史记录
😔 难受      →  可查看心情时间线
😫 非常烦    →  用于压力关联分析
```

#### 调整压力值
1. **拖动压力滑块**：0-100连续调整
2. **颜色反馈**：
   - 🟢 绿色 (0-20)：轻松
   - 🟡 黄色 (20-60)：中等
   - 🟠 橙色 (60-80)：较高
   - 🔴 红色 (80-100)：过高

#### 心情压力分析
1. 点击"**📊 分析**"按钮
2. 查看：
   - 7日心情变化趋势
   - 心情分布统计（各档占比）
   - 压力指数预测（未来3天）
   - 心情与压力的相关性分析

### 许可证管理

#### 激活许可证
```
1. 点击 "⭐ 升级" 或 "✓ 已激活" 按钮
2. 输入 "用户ID"（激活许可证时使用的ID）
3. 输入 "许可证密钥"（20个字符）
4. 点击 "激活"
5. 激活成功，所有功能解锁！
```

#### 查看许可信息
- 点击左上角 "✓ 已激活" 按钮
- 显示过期日期和剩余天数
- 点进去可实时查看天数倒数

#### 续期许可证
```
1. 在已激活状态点击 "续期" 按钮
2. 输入用户ID和新的许可证
3. 天数自动累加（旧的 + 新的）
4. 继续使用
```

## 管理员指南

### 生成许可证

#### 打开生成器
- 在浏览器中打开 `许可证生成器.html`

#### 生成步骤
```
1. 输入用户ID
   - 可以是：邮箱、学号、用户名等
   - 激活时必须使用相同的ID

2. 选择有效期
   - 30天
   - 90天
   - 180天
   - 365天
   - 730天（2年）
   - 36500天（永久）

3. 点击 "生成许可证"

4. 获得密钥格式：XXXX-XXXX-XXXX-XXXX-XXXX
```

#### 管理历史记录
- **复制**：点击"📋 复制"一键复制ID和密钥
- **导出Excel**：点击"📊 导出 Excel"下载历史记录
- **删除**：点击"🗑️ 删除"删除单条记录
- **清空**：点击"🗑️ 清空历史"删除所有记录

#### 分发许可证
```
复制 → 发送给用户
用户使用相同ID在应用中激活
```

## 常见问题

### Q: 演示版本有什么限制？
**A:** 演示版本（1天）禁用：
- 主题背景自定义（🎨 主题按钮禁用）
- 数据导出功能（导出按钮禁用）
- 其他功能完全可用

升级到专业版解除所有限制。

---

### Q: 许可证激活失败，提示"用户ID不匹配"？
**A:** 检查以下几点：
1. **用户ID必须完全相同**
   - 生成许可证时的ID
   - 激活时输入的ID
   - 必须完全一致（大小写敏感）

2. **许可证密钥正确**
   - 确保复制完整（24个字符）
   - XXXX-XXXX-XXXX-XXXX-XXXX

3. **许可证未过期**
   - 生成时间不能太早（检查系统日期）

如仍有问题，点击"🔍 诊断密钥"查看详细报告。

---

### Q: 如何续期许可证？
**A:** 
1. 确保当前已激活
2. 点击左上角"✓ 已激活"打开信息弹窗
3. 点击"🔄 续期"按钮
4. 输入用户ID（相同ID）和新许可证
5. 点击"激活"
6. 天数自动累加（旧期限 + 新期限）

---

### Q: 如何导出和备份我的数据？
**A:**
1. 点击左侧导航"💾 保存位置"
2. 点击"📥 导出数据"
3. 自动下载 JSON 文件，包含所有数据
4. 将文件保存在安全位置

**恢复数据：**
1. 打开应用
2. 点击"💾 保存位置"
3. 点击"📤 导入数据"
4. 选择备份文件
5. 所有数据恢复

---

### Q: 许可证信息弹窗的剩余天数在实时变化？
**A:** 是的！为了让你时刻了解剩余时间，剩余天数会实时计算并每秒更新一次。指导告诉该保留此项功能。

---

### Q: 永久许可证显示多少天？
**A:** 购买永久许可证后：
- 左上角显示："✓ 已激活 (永久)"
- 点进去的详情中也显示："永久"
- 永久许可证内部计算为36500天（100年）

---

### Q: 如何卸载或重置应用？
**A:**
1. **清除本地数据**：
   - 按F12打开开发者工具
   - 进入 Application → LocalStorage
   - 找到当前网站
   - 清除所有数据

2. **备份重要数据**：
   - 使用"📥 导出数据"先备份
   - 导出后再清除

3. **重新开始**：
   - 刷新页面
   - 将进入全新的演示版本

---

### Q: 我的数据会上传到云端吗？
**A:** 不会！所有数据完全本地存储在你的浏览器中：
- ✅ 数据只保存在你的电脑
- ✅ 不需要账户登录
- ✅ 不连接任何服务器
- ✅ 隐私完全保护

---

### Q: 可以在多个设备上使用同一个许可证吗？
**A:** 可以！许可证支持多设备激活：
1. **首次激活**：在设备A输入许可证
   - 绑定设备A的指纹
   
2. **使用另一个设备**：去设备B激活相同许可证
   - 可能提示"此许可证已在其他设备激活"
   - 点击"在此设备激活"确认
   - 将指纹切换到设备B

3. **同时使用多设备**：
   - 设备A和B都可以使用
   - 天数共享（一个许可证）
   - 推荐使用设备确认按钮避免冲突

---

### Q: 如何判断许可证是否过期？
**A:**
1. **已过期**：
   - 无法看到"✓ 已激活"按钮
   - 演示版标签出现
   - 部分功能禁用

2. **即将过期**：
   - 点击"✓ 已激活"查看精确日期
   - 提前续期避免中断

3. **永久有效**：
   - 显示"✓ 已激活 (永久)"
   - 无过期日期

---

### Q: 许可证生成器中的历史记录会保存多久？
**A:**
- 历史记录保存在生成器页面的 LocalStorage 中
- 永久保留，直到你手动删除
- 可随时导出为 Excel 备份
- 关闭浏览器不会丢失
