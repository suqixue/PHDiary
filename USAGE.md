## 一、快速开始

### 1.1 打开应用
在浏览器中打开：`PHDiary - 学术生涯进度与健康状态监测系统.html`

> 💡 首次打开自动进入**试用阶段**（7天），所有基础功能完全可用

### 1.1.1 如何激活许可证？

如果你已获取激活码，可以在试用期内或试用期后激活：

**激活步骤：**
1. 打开应用后，点击页面上的 **⭐ 升级许可证** 按钮
2. 在弹出的对话框中输入：
   - **用户名**：开发者分发给你的用户名
   - **激活码**：开发者分发给你的激活码（8位字符）
3. 点击 **激活** 按钮
4. ✅ 激活成功！许可证将保存到本浏览器，永久有效 ♾️

**激活特点：**
- ⚡ **无需网络**：完全离线激活（前端本地算法）
- 💾 **自动保存**：激活信息保存在浏览器本地存储
- 🔄 **跨会话有效**：关闭浏览器后再打开，许可证仍然有效
- 📱 **不同浏览器独立**：每个浏览器需要单独激活

> 🆘 **遗忘激活码？** 请联系开发者重新生成新的激活码。

### 1.2 设置个人信息
1. **点击左上角头像区域**
2. **修改个人信息**：

## 二、工作时间管理

### 2.1 基础工作打卡

**操作流程：**
1. 点击**"▶️ 开始工作"**按钮
2. 系统自动开始计时（显示实时秒数）
3. 点击**"⏸️ 暂停"**可临时停止计时
4. 点击**"⏹️ 结束工作"**保存工作记录

**数据保存：**
- 工作开始时间、结束时间、总时长自动记录
- 支持同一天内多次工作记录累计

### 2.2 工作效率指数计算

**公式：**

$$\text{工作效率指数} = \frac{\text{本周实际工作时长}}{\text{本周目标时长}} \times 100\%$$

其中：
- **本周实际工作时长** = 周一至周日累计工作时数
- **本周目标时长** = 系统默认40小时/周（可自定义）
- **100% = 目标达成**，超过100%显示"超额完成"

**出处：** 基于PomodoroTechnique和时间管理学理论

### 2.3 工作统计面板

在**"📊 数据统计"**中查看：
- **今日工作** - 本日累计时长
- **周累计** - 本周总工作时长
- **月累计** - 本月总工作时长
- **时间分布图** - 按小时显示工作分布

### 2.4 周报生成

**自动周报包含：**
- 本周工作总时长
- 工作效率评分
- 每日时间分布
- 完成任务统计
- 科研产出统计

> 📥 点击**"📥 导出周报"**下载为 Word/Excel 格式

## 三、心情与压力管理

### 3.1 心情记录

**5档心情等级：**
| 表情 | 心情等级 | 对应值 | 说明 |
|------|---------|-------|------|
| 😫 | 5 | 非常疲惫 | 极度压力、身心俱疲 |
| 😔 | 4 | 有点累 | 工作压力大、疲劳感强 |
| 😐 | 3 | 一般 | 正常工作状态 |
| 😊 | 2 | 良好 | 精力充沛、积极向上 |
| 😄 | 1 | 非常好 | 心情极佳、充满干劲 |

**操作：** 点击对应表情即可记录，支持一天多次记录

### 3.2 压力指数追踪

**压力等级定义：**
$$\text{压力指数} = 0 \text{ 到 } 100 \text{ 连续值}$$

- **0-30** 🟢 低压力：放松状态
- **31-50** 🟡 中等压力：正常工作压力
- **51-75** 🟠 高压力：需要调整
- **76-100** 🔴 极度压力：警告，需要缓解

**操作：** 拖动压力滑块（0-100）记录当前压力值

### 3.3 心情趋势分析

**计算公式：**

$$\text{周平均心情值} = \frac{\sum_{i=1}^{n} \text{每日心情值}}{n}$$

其中：
- 每日心情值 = 该天所有心情记录的平均值
- n = 记录的天数

**分析指标：**
- **心情动摇指数** = 本周最高值 - 最低值（值越小越稳定）
- **积极天数** = 心情值 ≤ 2 的天数（目标：60%以上）

**出处：** 参考心理学中的心境追踪方法（Mood Tracking）

### 3.4 情绪预警

系统自动监测：
- 🟡 **连续3天高压力**（压力 > 75）→ 显示"请注意身体"提示
- 🔴 **心情恶化趋势**（3天内心情值递增）→ 显示缓解建议

## 六、项目进度追踪

### 6.1 创建项目

**操作步骤：**
1. 点击**"🎯 新建项目"**按钮
2. 输入**项目名称**（如"Master学位论文"）
3. 设置**目标日期**（项目完成期限）
4. **可选：** 添加项目描述
5. 点击**"创建"**

### 6.2 项目进度更新

**两种更新方式：**

**方式1：手动调整**
- 拖动进度条至目标百分比
- 点击对应百分比快速设置
- 每次调整自动记录时间戳

**方式2：逐项完成**
- 将项目分解为多个子任务
- 子任务完成时自动计算进度
- 100%时自动更新主任务

### 6.3 自动完成分类

**系统特性：**
当项目进度达到 **100%** 时：
- ✅ **自动移至"已完成项目"分类**
- 📅 **记录完成日期**
- 🏆 **触发成就徽章解锁**（如"毕业论文完成"）
- 📊 **统计完成用时**

**完成用时计算：**
$$\text{项目完成用时} = \text{完成日期} - \text{创建日期}$$

### 6.4 项目统计

在数据面板查看：
- **进行中项目** 数量和列表
- **已完成项目** 数量和完成时间
- **项目完成率** = 已完成 / 总数 × 100%

## 七、运动与健康

### 7.1 运动记录

**操作步骤：**
1. 点击**"🏃 添加运动"**按钮
2. 选择**运动类型**（跑步/骑车/游泳/健身/瑜伽/其他）
3. 输入**运动时长**（分钟）
4. 输入**运动强度**（轻度/中度/高强度）
5. 点击**"记录"**保存

### 7.2 热量消耗计算

**公式（Mifflin-St Jeor方程）：**

$$\text{热量消耗} = \text{基础代谢率} \times \text{运动系数} \times \text{运动时长}$$

其中：
- **基础代谢率 (BMR)** = 静息状态下每小时消耗热量（系统根据性别/年龄预估）
- **运动系数**：
  - 轻度运动 = 1.1
  - 中度运动 = 1.3
  - 高强度运动 = 1.5

**出处：** Mifflin-St Jeor公式（国际通用热量计算标准）

### 7.3 健康指数计算

**综合指数公式：**

$$\text{周健康指数} = \frac{\text{运动时长占比} \times 40\% + \text{整体心情占比} \times 30\% + \text{工作均衡指数} \times 30\%}{3}$$

其中：
- **运动时长占比** = 本周运动时长 / 建议150分钟
- **整体心情占比** = 本周平均心情值 / 5
- **工作均衡指数** = 1 - |工作时长 - 40小时| / 40

### 7.4 健康追踪面板

显示内容：
- **本周运动次数**和总时长
- **预估热量消耗**（kcal）
- **健康指数评分**（1-100分）
- **运动类型分布**（饼图）
- **坚持天数**（连续运动天数）

#### 9.5.2 积分计算系统

**所有指标累加，公式如下：**

$$\text{总积分} = (\text{待办} \times 8) + (\text{打卡} \times 2) + (\text{番茄} \times 10) + (\text{科研} \times 20) + (\text{项目新建} \times 30) + (\text{项目完成} \times 60) + (\text{工作时间} \times 10)$$

**各项指标说明：**

| 指标 | 每单位EXP | 说明 | 计算方式 |
|------|----------|------|---------|
| ✅ **待办完成** | 8 | 完成一项任务 | 累计已完成待办数 |
| ⏱️ **工作打卡** | 2 | 每次工作打卡 | 累计所有工作session数 |
| 🍅 **番茄钟** | 10 | 完成一个番茄 | 累计所有番茄钟数 |
| 📚 **科研记录** | 20 | 记录一条科研工作 | 累计所有科研记录数 |
| 📋 **项目新建** | 30 | 创建一个新项目 | 当前进行中的项目数 |
| 🎯 **项目完成** | 60 | 完成一个项目 | 已完成项目总数 |
| ⏳ **工作时间** | 10/小时 | 每工作1小时 | 累计总工作小时数 |

**示例计算：**
假如你的数据为：
- 已完成待办：20项 → 20 × 8 = 160 EXP
- 工作打卡：150次 → 150 × 2 = 300 EXP
- 番茄钟：50个 → 50 × 10 = 500 EXP
- 科研记录：30条 → 30 × 20 = 600 EXP
- 进行中项目：5个 → 5 × 30 = 150 EXP
- 已完成项目：2个 → 2 × 60 = 120 EXP
- 工作时间：200小时 → 200 × 10 = 2000 EXP

**总EXP = 160 + 300 + 500 + 600 + 150 + 120 + 2000 = 3830 EXP** → 达到**Lv.5 实验狂魔**

## 十、常见问题

### Q1: 数据会丢失吗？
**A:** 所有数据实时保存到浏览器本地存储（localStorage），不会自动丢失。建议定期导出备份。

### Q2: 可以在多个浏览器/设备上使用吗？
**A:** 数据**仅保存在单个浏览器**，不同浏览器或设备的数据相互独立。需要同步时请导出再导入。

### Q3: 试用期过后怎样继续使用？
**A:** 可通过激活许可证延长使用期限，进入**"⚙️ 设置"** → **"许可证管理"**激活。

### Q4: 如何重置每日数据？
**A:** 点击**"更多"** → **"刷新每日数据"**，重置每日的心情/压力记录（保留历史数据）。

### Q5: 导出的数据格式是什么？
**A:** 支持Word、Excel、PDF三种格式，数据结构完整，可直接用于汇报或分析。

### Q6: 心情记录应该多久记一次？
**A:** 建议**每日至少1次**（上午/下午各一次最佳），这样能更准确反映心情变化趋势。

### Q7: 工作时长包括休息吗？
**A:** 不包括。点击**"暂停"**可停止计时进行休息，恢复后点击**"继续"**即可。

### Q8: 项目完成后还能编辑吗？
**A:** 可以。进度100%后自动移至"已完成项目"，点击仍可修改基本信息，但纪录的完成日期不变。

## 附录：数据分析公式汇总

### 时间管理指标
- **工作效率指数** = 实际工作时长 / 目标时长 × 100%
- **时间利用率** = 有效工作时长 / 在线时长 × 100%
- **平均日均工作** = 周工作总时长 / 7天

### 心理健康指标
- **周平均心情值** = Σ每日心情值 / 记录天数
- **心情动摇指数** = 周最高心情值 - 周最低心情值
- **压力峰值** = 周内最高压力值

### 健康指标
- **热量消耗** = BMR × 运动系数 × 时长（分钟）/ 60
- **周健康指数** = (运动时长占比×40% + 心情占比×30% + 工作均衡×30%) / 3
- **坚持指数** = 连续运动天数 / 目标天数 × 100%

### 任务管理指标
- **任务完成率** = 已完成任务 / 总任务数 × 100%
- **高优先级完成率** = 完成高优先级 / 总高优先级数 × 100%
- **项目完成周期** = 项目完成日期 - 项目创建日期

### 成就系统指标 ⭐ 新增
- **总积分(EXP)** = (待办完成×8) + (工作打卡×2) + (番茄钟×10) + (科研记录×20) + (项目新建×30) + (项目完成×60) + (工作时间小时×10)
- **当前等级** = 根据总EXP匹配8个等级中的对应等级
- **距离下级所需EXP** = 下一等级EXP阈值 - 当前总EXP
- **徽章解锁进度** = 已解锁徽章数 / 27个总徽章 × 100%
