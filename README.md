# PHDiary - 学术生涯进度与健康状态监测系统

面向硕博生的**一站式自我管理工具**，帮助你在高压科研中保持规律、高效与稳定。

**完全本地存储 · 无需登录 · 数据自有 · 离线可用**

## ✨ 核心功能

- 📊 **工作追踪** - 精确记录工作/休息时段，分析效率趋势
- 🎯 **项目管理** - 多层级子任务，灵活时间跟踪
- 📝 **学术记录** - 科研分类、日记、任务管理
- ✅ **待办清单** - 创建、编辑、完成任务，优先级和时间提醒
- 💪 **健康监测** - 运动、咖啡因、饮水、心情追踪
- 🏆 **成就系统** - 自动解锁徽章，可视化成长
- 📈 **数据分析** - 多维度图表，周报生成，数据导出

## 🚀 快速开始

1. **下载文件** - 仅下载`PHDiary - 学术生涯进度与健康状态监测系统.html`/也可下载整个项目文件
2. **打开应用** - 在浏览器打开 `PHDiary - 学术生涯进度与健康状态监测系统.html`
3. **设置信息** - 点击左上角头像修改个人信息
4. **开始使用** - 点击对应按钮开始记录工作、心情、任务等

> 💡 首次使用自动进入试用阶段（1天），所有基础功能完全可用 [联系作者即可获得永久激活码，完全免费]
