## 📖 使用指南速览

### 工作时间管理
```
1️⃣ 点击"▶️ 开始工作"开始计时
2️⃣ 工作中可进行"⏸️ 暂停"操作
3️⃣ 点击"⏹️ 结束工作"保存时段
4️⃣ 查看"📊 数据统计"中的工作效率评分
```
**关键公式：** 效率 = 实际工作时长 / 目标时长 × 100%

### 心情与压力追踪
```
1️⃣ 点击对应心情表情（😄 😊 😐 😔 😫）
2️⃣ 拖动压力滑块调整压力值（0-100）
3️⃣ 在"📅 心情记录"查看时间线
4️⃣ 使用"📈 分析"查看心情趋势
```
**心理指标：** 周平均心情值、心情动摇指数、情绪稳定性评价

### 任务完成管理
```
1️⃣ 点击"✅ 新建任务"创建任务
2️⃣ 选择优先级（🔴高/🟡中/🟢低）
3️⃣ 完成任务后打勾标记
4️⃣ 查看周任务完成率统计
```
**完成率目标：** 每周 ≥ 80%

### 项目自动完成分类
```
1️⃣ 创建项目并设置目标日期
2️⃣ 更新进度至100%
3️⃣ ✨ 自动移至"已完成项目"分类
4️⃣ 系统记录完成时间、用时周期
```
**新特性：** 项目完成自动分类，无需手动操作

### 运动与健康追踪
```
1️⃣ 点击"🏃 添加运动"记录运动
2️⃣ 系统自动计算热量消耗
3️⃣ 查看周健康指数评分
4️⃣ 获得运动成就徽章
```
**热量计算：** 基于Mifflin-St Jeor公式

### 数据分析与导出
```
1️⃣ 点击"📊 数据统计"查看各类图表
2️⃣ 系统自动生成周报
3️⃣ 支持Word/Excel/PDF导出
4️⃣ 数据备份定时保存
```
**导出格式：** Word(.docx) / Excel(.xlsx) / PDF
