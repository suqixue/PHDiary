## 📌 项目简介

**PHDiary** 是一个功能强大的个人时间管理、健康追踪和成就解锁系统，专为硕博生设计。无论你是在为论文砌砌碰壁、平衡工作与生活，还是想科学管理时间和身心健康，PHDiary 都能为你提供完整的解决方案。

### ✨ 核心特色

| 功能模块 | 描述 |
|---------|------|
| 🕐 **工作打卡** | 精准记录工作时间，分析工作效率，生成工作报表 |
| 💭 **心情管理** | 5档心情记录，压力指数动态追踪，智能情绪分析 |
| ✅ **任务管理** | 待办事项、优先级排序、截止提醒、完成统计 |
| 📚 **科研记录** | 文献阅读、论文撰写、实验数据、组会交流详细记录 |
| 🏃 **运动追踪** | 运动类型记录、卡路里消耗、健身趋势分析 |
| ☕ **咖啡因管理** | 咖啡摄入统计，健康预警，品牌库管理 |
| 💧 **饮水监测** | 每日饮水量记录，达成进度可视化，健康提醒 |
| 🏆 **成就系统** | 27个自动解锁徽章，个性化成就自定义 |
| 📊 **数据分析** | 多维度图表、周报生成、数据导出备份 |
| 🎨 **个性化界面** | 响应式设计、暗黑模式、自定义背景主题 |
| 🔐 **许可证系统** | 灵活的商业许可证管理，支持多设备激活 |

## ⚠️ 重要提示：数据备份建议

### 本地文件 LocalStorage 的限制

由于浏览器安全策略限制，通过 `file://` 协议打开本地HTML文件时，LocalStorage的行为存在不确定性：

**MDN官方说明：**
> "For documents loaded from `file:` URLs, the requirements for `localStorage` behavior are undefined and may vary among different browsers. In all current browsers, `localStorage` seems to return a different object for each `file:` URL. But there are no guarantees about that behavior, so you shouldn't rely on it because the requirements for `file:` URLs remain undefined. So it's possible that browsers may change their `file:` URL handling for `localStorage` at any time."

### 具体风险

1. **浏览器更新**：浏览器更新后可能改变file:// URL的LocalStorage处理方式
2. **隐私设置**：某些浏览器配置或隐私模式可能禁用LocalStorage
3. **跨浏览器不一致**：不同浏览器对本地文件的LocalStorage支持不同
4. **自动清理**：浏览器清理缓存时可能删除LocalStorage中的数据

### 🔥 强烈建议

**请定期手动导出和备份你的数据！**

#### 备份步骤
1. 打开应用 → 左侧"💾 保存位置"
2. 点击"📥 导出数据"
3. 自动下载 `PHDiary_备份_日期.json` 文件
4. **将备份文件保存在云盘或外部驱动器**

#### 推荐备份频率
- **每周备份一次** - 确保数据安全
- **重要数据变化后立即备份** - 如重要总结、关键成就等
- **定期检查备份文件** - 确保备份成功

#### 恢复数据
1. 打开应用 → 左侧"💾 保存位置"
2. 点击"📤 导入数据"
3. 选择备份文件 `PHDiary_备份_XXX.json`
4. 所有数据即刻恢复

## ❓ 常见问题

### Q: 我的数据会保存在哪里？
**A:** 所有数据保存在你的浏览器本地存储（LocalStorage）中，完全不上传到云端。

### Q: 许可证有什么用？
**A:** 许可证用于解锁所有功能（演示版部分功能受限）。购买许可证支持开发，让我们持续改进产品。

### Q: 可以在多个设备上使用同一个许可证吗?
**A:** 可以！许可证支持多设备激活。只需在每个设备上用相同的用户ID和许可证激活即可。

### Q: 许可证过期了怎么办？
**A:** 你可以随时续期。新许可证的天数会自动加到现有期限上。

### Q: 如何导出我的数据？
**A:** 点击左侧"💾 保存位置" → "📥 导出数据"，自动下载JSON备份文件。

### Q: 需要账户登录吗？
**A:** 不需要！完全零账户，无隐私泄露风险。
