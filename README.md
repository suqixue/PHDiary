## 📌 项目简介

**PHDiary** 是一个功能强大的个人时间管理、健康追踪和成就解锁系统，专为研究生设计。无论你是在为论文砌砌碰壁、平衡工作与生活，还是想科学管理时间和身心健康，PHDiary 都能为你提供完整的解决方案。

### ✨ 核心特色

| 功能模块 | 描述 |
|---------|------|
| 🕐 **工作打卡** | 精准记录工作时间，分析工作效率，生成工作报表 |
| 💭 **心情管理** | 5档心情记录，压力指数动态追踪，智能情绪分析 |
| ✅ **任务管理** | 待办事项、优先级排序、截止提醒、完成统计 |
| 📚 **科研记录** | 文献阅读、论文撰写、实验数据、组会交流详细记录 |
| 🏃 **运动追踪** | 运动类型记录、卡路里消耗、健身趋势分析 |
| ☕ **咖啡因管理** | 咖啡摄入统计，健康预警，品牌库管理 |
| 💧 **饮水监测** | 每日饮水量记录，达成进度可视化，健康提醒 |
| 🏆 **成就系统** | 27个自动解锁徽章，个性化成就自定义 |
| 📊 **数据分析** | 多维度图表、周报生成、数据导出备份 |
| 🎨 **个性化界面** | 响应式设计、暗黑模式、自定义背景主题 |
| 🔐 **许可证系统** | 灵活的商业许可证管理，支持多设备激活 |

## ❓ 常见问题

### Q: 我的数据会保存在哪里？
**A:** 所有数据保存在你的浏览器本地存储（LocalStorage）中，完全不上传到云端。

### Q: 许可证有什么用？
**A:** 许可证用于解锁所有功能（演示版部分功能受限）。购买许可证支持开发，让我们持续改进产品。

### Q: 可以在多个设备上使用同一个许可证吗?
**A:** 可以！许可证支持多设备激活。只需在每个设备上用相同的用户ID和许可证激活即可。

### Q: 许可证过期了怎么办？
**A:** 你可以随时续期。新许可证的天数会自动加到现有期限上。

### Q: 如何导出我的数据？
**A:** 点击左侧"💾 保存位置" → "📥 导出数据"，自动下载JSON备份文件。

### Q: 需要账户登录吗？
**A:** 不需要！完全零账户，无隐私泄露风险。

## 核心数据结构

系统使用以下主要数据对象进行管理：

```javascript
workData          // 工作和休息时段记录 (时间戳、状态)
moodData          // 心情和压力记录 (心情档位、压力值、时间)
intakeData        // 咖啡和饮水摄入记录 (品牌、类型、容量、时间)

exerciseData      // 运动记录 (运动类型、卡路里、日期)
researchData      // 科研工作记录 (类别、时长、日期)
profileData       // 用户个人信息 (姓名、学号、头像、压力值等)
todoData          // 待办任务 (标题、优先级、截止时间、完成状态)
diaryData         // 日记记录 (日期、内容、创建时间)
pomodoroData      // 番茄钟数据 (完成数、关联任务、日期)
projectData       // 项目数据 (名称、进度百分比、描述)
licenseData       // 许可证数据 (激活状态、过期时间、设备指纹)
BADGES            // 徽章配置数组 (27个徽章定义、条件、图标)
```
